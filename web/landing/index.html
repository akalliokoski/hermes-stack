Tailnet-only control plane

Hermes Stack landing page

This page collects the web UIs exposed from the VPS through Tailscale so they can be opened from the MacBook without needing a browser on the server itself.

Hermes Dashboard↗
Hermes Agent web UI for configuration, sessions, logs, and provider credentials.
Proxy target: 127.0.0.1:9119
Syncthing UI↗
Manage folder sync, approve devices, and inspect sync health for the shared Hermes workspace.
Proxy target: 127.0.0.1:8384
Hindsight UI↗
Browse and inspect the shared Hindsight service that backs long-term semantic memory for Hermes profiles. It is exposed on its own Tailnet HTTPS port because the app is not path-prefix safe.
Tailnet URL: :9443 → local 127.0.0.1:9999
Hindsight API↗
Raw API endpoint for Hindsight. Useful for diagnostics and direct integration testing over the tailnet.
Proxy target: 127.0.0.1:8888
Firecrawl API↗
Firecrawl service endpoint used by Hermes web tooling. Helpful for health checks and manual API exploration.
Proxy target: 127.0.0.1:3002
Access these URLs from any machine already joined to the same Tailscale tailnet. The canonical entrypoint is:
https://openclaw-vps.taild96651.ts.net/

All services remain bound to localhost on the VPS and are published outward only through Tailscale Serve.
