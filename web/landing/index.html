Tailnet-only control plane

Hermes Stack landing page

This page collects the web UIs exposed from the VPS through Tailscale so they can be opened from the MacBook without needing a browser on the server itself.

Hermes Dashboard↗
Hermes Agent web UI for configuration, sessions, logs, and provider credentials. It is exposed on its own Tailnet HTTPS port because the current dashboard frontend is not path-prefix safe.
Tailnet URL: :9444 → local 127.0.0.1:9119
Hermes Workspace V2↗
Optional zero-fork mobile/PWA workspace for Hermes Agent. It talks to the gateway's localhost-only API server, so you get chat, files, memory, and task views from a tailnet browser without exposing Hermes directly to the public Internet.
Tailnet URL: :9446 → local 127.0.0.1:3000
Syncthing UI↗
Manage folder sync, approve devices, and inspect sync health for the shared Hermes workspace. It is exposed on its own Tailnet HTTPS port because Syncthing rejects the proxied Tailnet hostname when served under the landing page path.
Tailnet URL: :9445 → local 127.0.0.1:8384
Hindsight UI↗
Browse and inspect the shared Hindsight service that backs long-term semantic memory for Hermes profiles. It is exposed on its own Tailnet HTTPS port because the app is not path-prefix safe.
Tailnet URL: :9443 → local 127.0.0.1:9999
Audiobookshelf↗
Self-hosted audiobook and podcast server for Hermes-generated episodes. Use this UI to finish first-run setup, manage libraries, and inspect the podcast feed used by iPhone and Apple Watch clients.
Tailnet URL: :13378 → local 127.0.0.1:13378
Jellyfin↗
Self-hosted video library for Hermes-generated explainers. Use it to create a library rooted at /media/videos and browse Manim-rendered NotebookLM-style videos over the tailnet.
Tailnet URL: :8096 → local 127.0.0.1:8096
Hindsight API↗
Raw API endpoint for Hindsight. Useful for diagnostics and direct integration testing over the tailnet.
Proxy target: 127.0.0.1:8888
Firecrawl API↗
Firecrawl service endpoint used by Hermes web tooling. Helpful for health checks and manual API exploration.
Proxy target: 127.0.0.1:3002
Access these URLs from any machine already joined to the same Tailscale tailnet. The canonical entrypoint is:
https://this-node.ts.net/

All services remain bound to localhost on the VPS and are published outward only through Tailscale Serve.
